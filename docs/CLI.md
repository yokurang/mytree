## `rtree`

Rtree lets you view directory trees with optional depth control, hidden files, extension filtering, and long-format metadata.

**Usage:** `rtree [OPTIONS] [PATH]`

###### **Arguments:**

* `<PATH>`

  Default value: `.`

###### **Options:**

* `-d`, `--depth <MAX_DEPTH>`
* `-a`, `--all`
* `-e`, `--extension <EXTENSIONS>`
* `-l`, `--long`
* `-o`, `--output <OUTPUT>`
* `--pager`
